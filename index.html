How it works

Three steps from zero to a working paid tool call.

1

Fund a wallet

Drop USDC and a little ETH into any wallet on Base. That's the buyer's tab.

2

Install the MCP

Add the tool's MCP server to Claude Desktop, Cursor, or Windsurf. Paste your wallet key into the config.

3

Let the agent call

Ask your agent to do the thing. The MCP signs the payment, the server settles over x402, you get the result.

Built for developers

Open-source clients, public seller code, no vendor lock-in. If you want to host your own seller for a tool, the patterns are reusable.
